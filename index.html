    注意事项

    因为如果实现好了一个类，想编译运行，记得把文件加到 CMakeList 文件的 SRC_LIST 里（现在为了保证编译成功只有 main 函数在源文件 list 里

    命名规范

    • 类名/文件名 - 帕斯卡（Pascal）命名法 ReplacePart.cpp
    • 成员变量名 - m_replacePart
    • .h 文件要加 #ifndef 等等

    如何编译运行可执行文件？

    mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug -G "Visual Studio 17 2022" -A x64
MSBuild ALL_BUILD.vcxproj /t:build /p:Configuration=Debug
./bin/Debug/GoldMiner.exe
